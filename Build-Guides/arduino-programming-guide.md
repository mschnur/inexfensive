# Setting Up The Project To Program The Arduinos

## Required Tools
* 1 Arduino Uno
* 1 Arduino Mega
* Computer
* Arduino USB Cable (USB Type A to USB Type B; Male/Male) [[Example](https://www.sparkfun.com/products/512)]
